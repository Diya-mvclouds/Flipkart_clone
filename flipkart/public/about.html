About Us

Welcome to Flipkart, a project designed to demonstrate a full-stack e-commerce application using HTML, CSS, JavaScript, Node.js, and MySQL.

Our Mission: To provide a seamless online shopping experience with a user-friendly interface and robust backend functionality.

Who We Are

We are a team of developers passionate about building scalable web applications. This replicates the core functionality of a major e-commerce platform, including user authentication, product management, cart systems, and order processing.

What We Offer

  • Wide Selection: Browse through various categories of products.
  • User Security: Secure login and data protection using JWT tokens.
  • Fast Performance: Optimized SQL queries for quick data retrieval.
  • Responsive Design: Shop comfortably on any device, mobile or desktop.

Our Technology Stack

This project is built with modern technologies:

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Node.js, Express.js
  • Database: MySQL
  • Authentication: JWT (JSON Web Tokens)
